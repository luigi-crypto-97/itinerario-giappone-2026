# Giappone 2026 — Guida di viaggio 🇯🇵

Guida completa e personalizzata per 15 giorni fra **Osaka, Kyoto e Tokyo** (8–23 agosto 2026):
itinerario giorno per giorno, mappe interattive, tips, budget, frasi con pronuncia, curiosità e checklist.

**Guida online:** dopo aver attivato GitHub Pages sarà su `https://<tuo-username>.github.io/giappone-2026/`

## Pubblicare
1. Crea un repo pubblico `giappone-2026` su GitHub.
2. Carica il contenuto di questa cartella (o `git push`).
3. Settings → Pages → Deploy from a branch → `main` / root → Save.

## Push da terminale
```
cd giappone-2026
git remote add origin https://github.com/<tuo-username>/giappone-2026.git
git push -u origin main
```
Tutto è in un unico file `index.html`. Le mappe Google si caricano dal browser dei visitatori. Nessun dato sensibile.
